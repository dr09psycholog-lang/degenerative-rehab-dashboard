Ключевые факторы прогноза
Влияют на траекторию реабилитации
Клинические
  • Время старта Чем раньше — тем лучше: окно нейропластичности
  • Мультидисц. Команда: невролог, психолог, эрготерапевт, соц.работник
  • Стадия Mild NCD → лучший ответ; тяжёлая деменция → паллиатив
  • Тип деменции ЛВД и ДТЛ: ограничения нейролептиков; БА: АХЭ
  • Семья Высокий уровень вовлечённости → +20–35% QoL пациента
  • Коморбидность ССЗ, сахарный диабет ускоряют прогрессирование

Caregiver burnout prevention
  • Психообразование Школа ухода — снижение burnout на 40%
  • Группы поддержки Очные и онлайн (Альцрус, Помощь рядом)
  • CBT для семьи Тревога, депрессия у ухаживающих
  • Технологии Телемедицина, GPS-трекеры, напоминалки лекарств
  • Передышка (Respite) Дневной стационар — «передышка» для семьи
