## 📋 Содержание дашборда

- **KPI-блок** — охват реабилитационных программ, вовлечение семей, снижение BPSD, QoL
- **Нозологическая матрица** — Болезнь Альцгеймера, Пика/ЛВД, ДТЛ, Паркинсона, органические поражения
- **Немедикаментозные методы** — CST, LSVT BIG, музыкотерапия, rTMS (EBM Grade A–B)
- **Поддержка семьи и НКО** — Россия и международный уровень
- **Фильтры** — нозология, стадия (Mild NCD / Деменция), фокус (пациент / семья / НКО)
